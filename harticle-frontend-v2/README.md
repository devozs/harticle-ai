# Harticle Frontend v2 (Nuxt 4)

Nuxt 4 rewrite of the Harticle UI. Requires **Node 22**.

## Setup

```bash
nvm use 22
npm install
```

## Development

```bash
npm run dev
# http://localhost:3000 (or pass --port 3000)
```

Environment (optional):

- `NUXT_PUBLIC_API_BASE` — default `http://localhost:8080/api` in dev
- `NUXT_PUBLIC_API_HOST` — default `http://localhost:8080`
- `NUXT_PUBLIC_SITE_URL` — default `http://localhost:3000`

## Build

```bash
npm run build
npm run preview
```
